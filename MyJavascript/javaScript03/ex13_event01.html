** Event Basic **
    => 이벤트속성, 이벤트리스너  : onclick
      -> on 을 제외한부분을 이벤트이름, 이벤트타입 이라함.
    =>  이벤트핸들러
      -> 이벤트속성에 대응되는 함수   
      -> onclick="start()"
    => 이벤트모델
      -> dom객체 (Tag)의 이벤트리스너 와 이벤트핸들러 를 연결하는 방법
      -> 1)인라인  2)인라인+스크립트  3)고전적(Traditional) 방식
