** 내장함수 eval 을 이용한 계산기 **

=> eval : 문자열(3+5*2)을 수식으로 바꾸어서 계산을 해줌
=> eval("3+5*2") -> 연산결과 13 제공

// ** eval 함수 // => 문자로 표현 된 JavaScript 코드를 실행하는 함수 // => return(반환) 값 : 매개변수로 전달된 코드(수식)를 평가한 결과값. (값이 없으면 undefined 반환) // => 위험성 : https://velog.io/@modolee/javascript-eval-is-evil // .eval()은 인자로 받은 코드를 caller의 권한으로 수행하는 위험한 함수 // .악의적인 영향을 받았을 수 있는 악의적인 코드를 수행하는 결과를 초래할 수 있음. // . 제3자 코드가 eval()이 호출된 위치의 스코프를 볼 수 있으므로 악의적인 공격이 가능 // . 최신 JS 엔진에서 여러 코드 구조를 최적화하는 것과 달리, // eval()은 JS 인터프리터를 사용하기 때문에 느림 // ( 최신 자바스크립트 인터프리터는 코드를 기계 코드로 변환함. 즉, 변수명의 개념이 완전히 사라짐 // eval()을 사용하면 브라우저는 기계 코드에 해당 변수가 있는지 확인하고 값을 대입하기 위해, // 길고 무거운 변수명 검색을 수행해야 함 // eval()을 통해 자료형 변경 등 변수에 변화가 일어날 수 있으며, // 브라우저는 이에 대응하기 위해 기계 코드를 재작성해야 함 )
