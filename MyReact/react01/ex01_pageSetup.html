** React 의 기본 구동원리
     => 가상DOM 
     리액트 앨리먼트로 이루어진 DOM 
     리액트는 효율적 SPA를 위해 브라우저 DOM 을 갱신해주기위한 라이브러리 이고 
     이를 위해 직접 DOM 을 다루는것보다 가상DOM 을 다루는것이 더 편리함.
                
     => 리액트 앨리먼트 만들기 : React.createElement(엘리먼트타입,엘리먼트속성, 컨텐츠)
        
     => 브라우저에 랜더링 : ReactDOM.render(랜더링할_엘리먼트, 랜더링대상_DOM_노드)
